# MeshUp: Image to Plastic Canvas Converter

A browser-based utility for converting digital images into needlepoint patterns specifically for plastic canvas.

Features

Smart Pixelation: Adjust the grid density to match your project size.

Mesh Standards: Presets for 7, 10, and 14-count plastic canvas.

Real-time Stats: Calculates physical dimensions based on stitch count.

Color Legend: Automatically identifies a yarn palette from your image.

Counting Grid: Bold markers every 10 stitches for easy tracking.
